NFL Table

Featured: NFL-Table; Description: Single-page Web Application for interactively querying, filtering, and analyzing play-by-play NFL statistics over the past ~10 seasons. Features: Preserve application state (form data, table display, etc.) into a shareable URL. Javascript-based metadata layer acts as a "glue" to the front and back-end. Adding a new field or filter to both the database and the UI is accomplished in one central repository. Front-end: React, Ant Design Framework; Back-end: Node/Express; Google BigQuery; Google Firestore; Cloud Technologogies: Firebase Hosting, Firebase Functions (Serverless)
